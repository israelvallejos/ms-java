About This Microservice

Congratulations on creating a new Quarkus microservice! This microservice provides several endpoints to retrieve mock data for various purposes.

Available Endpoints

  • /weather - Get mock weather data.
  • /currency - Get mock currency values.
  • /presidents - Get a list of USA presidents.
  • /advice - Get random life advice.
